Q3. 	"Guess the output

            var a = 10;
            (function (){
                console.log(a);
                var a = 20;
            })();"

In this Anonymous function, a is being redeclared inside the function scope so a new memory will be allocated for a inside function scope during memory allocation. And at the time of execution console line will run before the value of a is assigned. So Hoisting will be there and the Output will be UNDEFINED
