 Q4.	"Print output: 
            var x = 21;
            var girl = function () {
                console.log(x);
                var x = 20;
            };
            girl ();"

In this function, x is being redeclared inside the function scope so a new memory will be allocated for x inside function scope during memory allocation. And at the time of execution console line will run before the value of x is assigned. So Hoisting will be there and the Answer will be UNDEFINED 

Q5.	"Print output: 
            var x = 21;
            girl ();
            console.log(x)
            function girl() {
                console.log(x);
                var x = 20;
            };
            "

Inside global memory allocation, x and the function girl will be declared and at the time of execution girl function will be executed first and give answer UNDEFINED due to Hoisting inside function girl and after that console line will be executed and give answer 21

Q.6	"Print output

            var x = 21;
            a();
            b();
            
              function a() {
                
               x = 20;
              console.log(x);
            };
             function b() {
                
                x = 40;
               console.log(x);
            };"

Output is 20 and 40 First of all at the time of memory allocation in global scope x and function a and function b will be stored and for global execution value of x will be initialized 21 then function a will be called inside which value of x will be re-initialized as 20 and give output as 20 . Then function b will be called inside which value of x will again be re-initialized as 40 and output will be 40

Q7. Write a function that accepts parameter n and returns factorial of n

function factorial(n){
            if(n < 0){
                console.log(`Error: The value of n can not be zero`);
            }
            if(n === 0){
                console.log(`The factorial of 0 is : 1`);
            }
            else{
            let fact = 1;
                
                for(let i=1; i<=n ; i++){
                    fact *= i;
                }
                console.log(`The factorial of ${n} is ${fact}`);
            }
         }
         factorial(4);

Output for value of n = 4 will be 24
